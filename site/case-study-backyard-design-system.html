Overview


Backyard is Lowe's enterprise design system, used across product teams in the US and India. DS4 was the fourth generation — a significant architectural evolution introducing design tokens, improved component structure, and tighter Figma-to-code alignment.

I joined the DS4 initiative as Figma SME — responsible for Figma architecture decisions, component library structure, token implementation, and design team onboarding.


The fourth generation of Backyard — a complete architectural rebuild of Lowe's enterprise design system.

The Challenge

Inconsistent component structures Token naming misaligned with engineering Fragmented handoff documentation Low designer adoption

DS3 had accumulated technical debt. Component structures were inconsistent, token naming was not aligned with the engineering token system, and documentation was fragmented. DS4 was the opportunity to rebuild this foundation correctly — so that designers actually used the system rather than worked around it.

My Contribution

Figma architecture Token system design Component library Documentation Team enablement

I led the Figma side of DS4 — designing the token architecture (primitives, semantics, component tokens), building and documenting the component library with consistent naming aligned to engineering, authoring usage documentation, and running onboarding sessions with the design team.

I also served as the go-to resource for Figma-related questions across the India design team, troubleshooting issues and ensuring designers could work confidently within the new system.


Three-tier token architecture: primitive values → semantic aliases → component-level tokens, aligned 1:1 with engineering implementation.

The Result


Token alignment between Figma and engineering eliminated a category of handoff errors entirely.

Designers adopted DS4 faster than previous system versions, supported by documentation and enablement that made the system approachable. The component library became the single source of truth for product design work across the India team.
